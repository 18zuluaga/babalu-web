Proyecto Babalu Web (Next.js + Tailwind)

Comandos

- dev: npm run dev
- build: npm run build
- start: npm run start

Despliegue en Vercel

1. Inicia sesión en Vercel y crea un nuevo proyecto.
2. Importa el directorio `/home/matias/babalu/babalu-web` desde tu repo.
3. Usa Node 20. Vercel lo detecta automáticamente.
4. Build Command: `npm run build`, Output: `.next`.

Estructura

- `src/app`: Rutas (Inicio, DOFA, PESTEL, SMART, etc.)
- `src/components`: `Navbar`
- `public`: Imágenes extraídas del DOCX
